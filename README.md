# Домбаза — лендинг

Одностраничный сайт (`index.html`) для магазина товаров для дома и ремонта «Домбаза».

## Содержание страницы

- **Название компании:** Домбаза
- **Описание/слоган:** «Всё для дома и ремонта» — инструмент, сантехника, электрика и отделочные материалы с доставкой по всей России
- **Товарные категории (4):**
  - Инструменты — ручной и электроинструмент, от отвёртки до перфоратора
  - Сантехника — трубы, смесители и фитинги для водоснабжения и отопления
  - Электрика — кабель, розетки, автоматы и освещение
  - Отделочные материалы — краска, плитка, ламинат и всё для чистовой отделки
- **Контакты / связь с клиентом:** форма заявки (имя, телефон, что заказать) в разделе «Оставить заявку» — компания перезванивает клиенту после отправки заявки в течение рабочего дня

## Структура HTML-документа (`index.html`)

1. `<head>` — `<title>Домбаза — товары для дома и ремонта</title>`, `<meta name="description">` с кратким описанием магазина
2. `<header class="site-header">` — логотип «Домбаза» и кнопка «Оставить заявку»
3. `<section class="hero">` — заголовок и подзаголовок с сутью предложения, кнопка перехода к форме
4. `<section id="categories">` — блок «Что у нас есть»: 4 карточки категорий товаров с иконками и описанием
5. `<section class="band-blue">` — блок «Почему выбирают Домбазу»: 4 преимущества (5 лет на рынке, доставка за 1–2 дня, возврат без проблем, скидки постоянным клиентам)
6. `<section id="delivery">` — блок «Доставка»: интерактивный выбор зоны (Москва и область / Санкт-Петербург / остальная Россия), для каждой зоны показывается цена и срок доставки
7. `<section id="order">` — блок «Оставить заявку»: форма (имя, телефон, что заказать) с подтверждением после отправки
8. `<footer>` — название компании и копирайт

## Технические детали

- Один файл, без внешних зависимостей кроме шрифтов Google Fonts (Oswald, PT Sans, PT Mono — с поддержкой кириллицы)
- Адаптивная вёрстка, доступность с клавиатуры, уважение `prefers-reduced-motion`
- Форма отправки — мокап (визуальное подтверждение на странице, без реальной отправки данных)
